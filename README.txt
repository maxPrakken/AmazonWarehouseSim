Note voor het opstarten van de simulatie.
------------------------------------------
Soms kan het zijn dat er een error ontstaand bij het starten van de simulatie.
Deze kon helaas niet meer opgelost worden voor de deadline.

Om deze error te voorkomen is er een "work around" die ervoor zorgd dat deze error
niet voorkomt.

1. Open het project.

2. Open Robot.java.

3. Zet een breakpoint op lijn 75 |     if(destNodes.size() > 0) {

4. Run de simulatie tot het op de breakpoint komt.

5. refesh de webpagina.

6. Haal de breakpoint weg.

7. Hervat het programma.

Op de manier wordt de error niet tegengekomen.
